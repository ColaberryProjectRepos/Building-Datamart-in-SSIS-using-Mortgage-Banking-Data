Creating a DataMart using Star schema Modelling 3

·         Based on Tables we have, Star Schema modelling with 1 Fact and 4 Dimension Tables.

·       The star schema separates business process data into facts, which hold the measurable, quantitative data about a business, and dimensions which are descriptive attributes related to fact data. Examples of fact data include Loan Amount, Terms, Down Payment, Loan Remaining. Related dimension attribute examples include, Loan Status, Mortgage Funded Date, Owner’s First Name, Last Name, Property Name, Region and so on.

 ·       Create a new Database (Mortgage Datamart) to load the Fact and Dimension Tables using the script in Code file

·       Right click or Double click on the Code File downloaded.


·         Then the code will appear in SQL Server Management Studio (SSMS).

·         Click on Execute above the Object Explorer.


·         New Database [Mortgage Datamart] created successfully


·         Right click on Database --- > Refresh. Then newly created Database will appear in Object Explorer.
